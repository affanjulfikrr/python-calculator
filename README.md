# Python Calculator

একটা সিম্পল ক্যালকুলেটর প্রোগ্রাম Python দিয়ে তৈরি করা। যোগ, বিয়োগ, গুণ, ভাগ — সবই করা যায়।

আমার প্রথম ছোট প্রজেক্ট 😄

## Features
- যোগ (+)  
- বিয়োগ (-)  
- গুণ (×)  
- ভাগ (÷)  
- সুন্দর করে আউটপুট দেখায়

## কীভাবে চালাবে

1. Python 3 ইনস্টল করে নাও (যদি না থাকে)
2. এই রিপোজিটরি ক্লোন করো বা ডাউনলোড করো
